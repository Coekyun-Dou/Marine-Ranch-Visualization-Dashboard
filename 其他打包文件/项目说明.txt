📋 项目信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
项目名称：智慧海洋牧场可视化平台
项目类型：大学生计算机设计大赛 - 数据可视化赛道
开发时间：2024年
技术栈：Vue 3 + Cesium + ECharts + Node.js + Socket.IO


🎯 核心功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ 3D地球可视化 - 基于Cesium的三维地球展示牧场位置
✓ 视频监控系统 - 水下摄像头实时监控网箱情况
✓ 数据实时分析 - 实时采集并展示水温、pH、溶氧量等环境数据
✓ 鱼类分布统计 - 饼图展示不同鱼类的养殖分布情况
✓ WebSocket实时通信 - 毫秒级数据更新


📁 项目结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

├── start.bat         # 一键启动脚本（Windows）
├── stop.bat          # 停止服务脚本（Windows）
├── README.md         # 详细项目文档
└── 快速启动指南.md  # 快速上手指南


🚀 快速启动（3步）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 确保已安装 Node.js（16.x或更高版本）
2. 双击运行 start.bat（首次运行会自动安装依赖）
3. 等待浏览器自动打开，访问 http://localhost:5173

⚙️ 配置说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
传感器API地址配置：
  文件：backend/server.js
  字段：sensorAPIAddress: 'SensorAPIAdress001'
  说明：当前使用占位符，实际使用时需替换为真实传感器API

端口配置：
  后端端口：3000（可在 backend/server.js 修改）
  前端端口：5173（可在 frontend/vite.config.js 修改）
