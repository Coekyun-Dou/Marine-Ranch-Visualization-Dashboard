# 智慧海洋牧场可视化平台（大计赛展示说明）

## 一、项目定位
本项目面向“海洋牧场数字化管理与智能化养殖”场景，构建集**数据可视化、实时监控、智能分析与鱼类识别**于一体的综合展示平台。通过多源传感数据、水文数据、鱼群监控数据的统一展示，帮助管理者实现环境评估、风险预警与辅助决策。

## 二、核心亮点
1. **一屏统览**：综合监控大屏将水质、水文、鱼群、地理位置等数据在同一界面展示。
2. **实时联动**：前后端通过 Socket.IO 提供实时数据推送与动态更新。
3. **AI 辅助**：AI 助手融合当前环境数据，自动给出养殖建议与风险提示。
4. **鱼类识别**：支持上传图片/视频进行识别，返回带框标注结果和统计信息。
5. **可扩展**：后端预留传感器接入逻辑与设备数据接口，方便接入真实业务。

## 三、系统架构
- **前端**：Vue3 + Vite + Element Plus + ECharts
- **后端**：Node.js + Express + Socket.IO
- **识别模型**：YOLOv8（Python 推理）
- **数据来源**：模拟传感器数据 + 可接入真实感知层

## 四、主要功能模块
1. **综合监控大屏**：展示水质、鱼群、设备、位置等关键指标。
2. **水质监控**：实时水温、pH、溶氧、盐度、浊度曲线和历史趋势。
3. **水文监测**：潮位、风速、波浪、海流等水文信息与统计图表。
4. **鱼群监控**：摄像头模拟 + 上传图片/视频识别 + 结果可视化。
5. **AI 助手**：基于当前传感数据的智能分析与对话式建议。

## 五、鱼类识别展示流程（PPT重点）
1. 进入“鱼群监控”页面。
2. 点击“导入图片/视频分析”。
3. 后端调用 YOLOv8 识别，生成**带框标注图片/视频**。
4. 前端展示识别结果并同步更新统计信息。
