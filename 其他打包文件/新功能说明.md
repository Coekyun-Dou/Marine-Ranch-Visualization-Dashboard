### 3. 🤖 新功能：AI 管理助手
**访问地址**：http://localhost:5173/ai-assistant  
**接入API**：DeepSeek AI（专业海洋养殖助手）

#### 功能特点：
1. **智能对话**
   - 分析水质、水文监测数据
   - 提供养殖管理建议（投喂、换水、增氧等）
   - 预警异常情况，给出应对措施
   - 回答海洋养殖相关专业问题

2. **快捷功能**
   - 🔍 分析当前水质
   - 🌤️ 天气影响预测
   - 🐟 投喂建议
   - 💡 养殖优化建议

3. **数据上传分析**
   - 支持上传 CSV/Excel 监测数据
   - 自动解析并生成分析报告
   - 给出具体的改善建议

4. **实时环境监测**
   - 左侧面板显示当前环境数据
   - 右侧面板显示数据分析摘要
   - 智能建议根据数据动态生成

5. **专业分析**
   - 水温、pH、溶氧量智能评估
   - 自动判断环境状态（正常/警告/危险）
   - 提供可操作的具体建议

#### 使用方法：

**方式1：通过导航栏访问**
- 在任何页面顶部导航栏点击"AI助手"

**方式2：直接访问**
- http://localhost:5173/ai-assistant

**方式3：从水文监测页面跳转**
- 水文监测页面右上角"AI助手"按钮

#### 对话示例：

```
用户：请分析当前的水质数据
AI：根据当前监测数据：
    - 水温 23.5°C，处于适宜范围（20-26°C）
    - pH值 8.1，正常范围内
    - 溶氧量 7.8mg/L，充足
    
    整体评估：水质状况良好，适合正常养殖活动。
    建议：保持当前管理策略，继续定期监测。
```

```
用户：今天应该投喂多少？
AI：根据当前环境数据分析：
    - 水温 23.5°C，鱼类活动活跃
    - 溶氧充足，消化能力强
    
    建议：可按正常投喂量的 100% 投喂
    投喂时间：上午 8-9点，下午 3-4点
    注意事项：分2-3次投喂，每次15-20分钟内吃完为宜
```

## ⚠️ 注意事项

1. **AI 功能需要网络连接**
   - DeepSeek API 需要互联网访问
   - 如果网络不佳，会自动使用本地备用分析

2. **数据导出**
   - 导出的数据为模拟数据
   - 实际部署时会替换为真实传感器数据

3. **浏览器兼容性**
   - 推荐使用 Chrome、Edge 浏览器
   - 确保浏览器版本较新

## 💡 常见问题

### Q1: AI 助手回复很慢？
**A**: 正常情况下 3-10 秒，如果超过 30 秒可能是网络问题。

### Q2: 数据导出后中文乱码？
**A**: 用 Excel 打开时，选择"数据" → "从文本" → 编码选择 UTF-8。
     或直接用 Excel 2016+ 打开，会自动识别。

### Q3: AI 助手无法访问？
**A**: 检查：
   1. 后端服务是否正常启动
   2. 浏览器控制台是否有错误
   3. 网络连接是否正常

### Q4: 想要更改 AI 的回复风格？
**A**: 可以修改 `backend/server.js` 中的 `systemPrompt`。
